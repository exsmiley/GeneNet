About

Welcome to Gene Net! This is a web app designed to enhance gene-targeted therapies by visualizing the intricate connections between around 3000 genes. Gene Net offers users an interactive visual experience with genes in their networks along with a database that allows them to personalize their searches.

Resources

Human gene network data was pulled from TRRUST (http://www.grnpedia.org/trrust/). PMIDs for each gene were obtained from this data and then used in PubMed in order to scrape additional data, such as the official name and symbol. *Brainstorming* Sigma visualization Bootstrap json JavaScript Python
